Security
Avoid sqlalchemy.text() with dynamically constructed SQL strings as this bypasses SQLAlchemy's SQL injection protections. The constructed SQL is passed to the database mostly unchanged, making it vulnerable when user input is involved. Replace f-strings/concatenation with named parameters like `text("... :param_name")` and pass parameters separately to execute().
