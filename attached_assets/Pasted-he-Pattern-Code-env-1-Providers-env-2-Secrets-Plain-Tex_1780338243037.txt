he Pattern: Code → env_1 (Providers) → env_2 (Secrets)
Plain Text
┌─────────────────────────────────────────────────────────────┐
│  CODE FILES (functions)                                      │
│  e.g., send_email(), store_blob(), query_db()               │
│  These NEVER know which provider they're using              │
│  They import from a mount point that env_1 signals          │
└──────────────────────┬──────────────────────────────────────┘
                       │ reads provider config
                       ▼
┌─────────────────────────────────────────────────────────────┐
│  env_1 (PROVIDERS) — SHIPS TO GIT ✅                        │
│  Maps each function to its provider implementation          │
│  e.g., EMAIL_PROVIDER=azure_communication_services          │
│        STORAGE_PROVIDER=azure_blob                          │
│        DB_PROVIDER=cosmos_db                                │
│  Swap providers by changing ONE file, not 50 code files     │
└──────────────────────┬──────────────────────────────────────┘
                       │ handshake (validates keys exist)
                       ▼
┌─────────────────────────────────────────────────────────────┐
│  env_2 (SECRETS) — NEVER IN GIT ❌                          │
│  Actual keys, connection strings, tokens                    │
│  Lives in:                                                  │
│    • AWS Secrets Manager                                    │
│    • Azure Key Vault                                        │
│    • Google Secret Manager                                  │
│    • LOCAL: symio_3.0_sidecar/.env (symlinked to repo)     │
│  The symlink means the repo .env → sidecar/.env            │
│  .gitignore blocks the .env, sidecar is outside repo       │
└─────────────────────────────────────────────────────────────┘

The Handshake Protocol:
1.env_1 declares: "I need EMAIL_API_KEY, EMAIL_ENDPOINT, DB_CONNECTION_STRING"
2.env_2 emits handshake: validates all required secrets exist before app boots
3.If any secret is missing → hard fail at boot with clear error (not a runtime crash 3 hours later)
The Symlink Safety:
Plain Text
/home/user/repos/DevGov/.env → symlink → /home/user/symio_3.0_sidecar/.env.devgov

•.env is in .gitignore (can never be committed)
•The actual file lives in the sidecar directory (completely outside the git tree)
•Even if someone removes .gitignore, the symlink target is outside the repo
